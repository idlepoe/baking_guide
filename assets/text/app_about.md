# 빵실기 - 제빵기능사

**제빵기능사 실기를 앱 하나로 연습하는 무료·무광고 가이드 앱**

빵실기는 제빵기능사 실기 시험을 준비하는 분들이, 교재와 타이머·계산을 오가며 헤매지 않도록 만든 연습용 앱입니다. 개발자가 직접 시험을 준비하며 필요했던 기능을 모아, 누구나 손쉽게 따라 할 수 있게 정리했습니다.

---

## 앱 소개

실기는 단계가 많고 시간 관리가 까다롭습니다. 빵실기는 **품목별 레시피를 단계 단위로 안내**하고, **발효·시험·굽기 등 타이머**와 **반죽 온도 계산**을 한곳에서 다루도록 설계했습니다. 여러 품목을 동시에 연습할 때도 **진행 탭**에서 한눈에 상태를 확인할 수 있습니다.

광고 없이 핵심 기능만 제공합니다. 연습에 집중할 수 있도록 UI는 단순하게, 내용은 시험 흐름에 맞게 구성했습니다.

---

## 주요 기능

### 레시피

- **품목 목록** — 시험 품목별 소요 시간, 난이도(별점), 썸네일 확인
- **북마크** — 자주 연습하는 품목을 저장하고, **최근 북마크 순**으로 목록 상단에 표시
- **실기 상세** — 단계별 제목, 설명, 이미지, 체크리스트, 감점 포인트 안내
- **레시피 요약** — 시험 정보·반죽·발효·성형·굽기 등 핵심 포인트 한눈에 보기
- **재료·배합** — 품목별 재료 목록, 배합 비율(배율) 조절로 계량 참고

### 실기 진행

- **단계별 진행** — 현재 단계 표시, 단계 진행 바, 좌우 스와이프로 단계 이동(설정에서 on/off)
- **체크리스트** — 단계마다 확인 항목 체크
- **동시 진행** — 여러 품목·세션을 동시에 진행하고 **진행 탭**에서 상태·경과 시간·예상 완료 시각 확인
- **진행 데이터** — 기기에 저장되어 앱을 다시 열어도 이어서 연습 가능

### 타이머

- **단계·품목별 타이머** — 발효, 시험 시간, 굽기 등 레시피에 맞춘 타이머 시작
- **홈 상단 진행 바** — 실행 중인 타이머를 앱 어디서나 빠르게 확인·이동
- **백그라운드 알림** — 완료 시 알림(소리·진동 설정 가능), 진행 중 알림 표시
- **타이머 알림 설정** — 전역 on/off, 알림 소리(기본/무음), 진동 on/off

### 반죽 온도 계산

- **물온도 계산기** — 믹서 종류·반죽 방식에 맞춘 마찰열을 반영해 목표 반죽 온도에 맞는 물 온도 안내

### 설정·편의 기능

- **다크 모드** · **앱 테마 색상** · **글자 크기** 조절
- **화면 자동 꺼짐 방지** — 연습 중 화면이 꺼지지 않도록 유지
- **모든 진행 데이터 초기화** — 저장된 진행·타이머 데이터 일괄 삭제

## 이렇게 사용해 보세요

1. **레시피** 탭에서 품목을 선택하고 북마크해 두세요.
2. 실기를 시작한 뒤 단계를 따라가며 체크리스트와 타이머를 활용하세요.
3. 물온도가 필요한 단계에서는 계산기를 참고하세요.
4. 다른 품목을 병행할 때는 **진행** 탭에서 전체 일정을 확인하세요.
5. **설정**에서 알림·화면·글자 크기를 본인 환경에 맞게 조정하세요.

## 개발 배경

이 앱은 개발자가 제빵기능사 실기를 준비하는 과정에서 직접 쓰기 위해 만들었습니다. 비슷한 고민을 하는 분들에게 조금이라도 도움이 되었으면 합니다.
